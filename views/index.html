SNES MARIO KART API

This API provides information about the SNES version of the classic game, Mario Kart. It allows users to retrieve data about various game elements such as characters, items, tracks...

/api/characters

https://www.snes-mk.onrender.com/api/characters

Has information about all playable characters in the game.

Return value: [character]
Param Required Description
name ‚ùå Name of the character you want information about.

*Requires mk-token header with corresponding key for authorization. Character property animatedImage gets auto-generated.

/api/characters/{id}

https://www.snes-mk.onrender.com/api/characters/1

Retrieve a single character by id.

Return value: character
Name Data Type
_id Number
name String
category String
stats Object
cpuItem Number
animatedImage String

*Requires mk-token header with corresponding key for authorization. Must include all fields except animatedImage which gets auto-generated.

*Requires mk-token header with corresponding key for authorization.
